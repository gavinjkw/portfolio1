Livable
www.livableroomates.com

(technologies used: Java, Spring Boot, MySQL, HTML5, CSS, Materialize) A platform for people living with roommates. Users are able to create a house and utilize a chat area, bill management, and event postings. Users can create a new house, or join a house with an access code.

Snow Trades
www.snowtrades.com

(technologies used: Python, Django, MySQL, HTML5, CSS, Bootstrap, Stripe API) A marketplace for used snow equipment where users can upload items for sale. Stripe has been integrated into the site, and users are able to upload pictures for item postings. An admin portal allows for control over the site and removal of users and items.
